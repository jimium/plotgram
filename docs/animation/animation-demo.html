Drawify SVG + CSS 动画效果演示

所有动画纯 CSS 实现，内嵌于 SVG，零 JS 依赖（页面控制按钮除外）

1. 节点入场动画

新增节点 → 弹性缩放 + 淡入。stagger 延迟依次出现，模拟 Patch Add。

2. 边数据流（自动循环）

stroke-dashoffset 循环动画，表达"调用方向"。纯 CSS，零 JS。

3. 节点位置平移

Layout Intent 微调时，节点从旧位置平滑移动到新位置。点击按钮触发。

4. 节点删除（淡出+缩小）

删除节点时，不是瞬间消失，而是平滑淡出缩小。点击按钮模拟。

5. Hover 交互 & 聚焦高亮

悬浮节点放大+阴影；点击"聚焦"按钮，其他元素淡化。纯 CSS。

6. 参数化指令（activate/highlight）

渲染参数指定 activate(node) → 脉冲发光；highlight(edge) → 加粗闪烁。

activate(active_node) + highlight(edge_to_db)

7. Steps 分步演示（纯 CSS :target 无 JS 控制）

点击按钮切换步骤。每个步骤是完整图，帧间用 Patch 动画过渡（这里演示淡入+移动，实际是 diff 自动生成）。
