Drawify Logo — 17 号「嵌套」系列变体

基于嵌套概念（外框+内核 = 多层抽象）的 10 组优化方向

17-A
精修原版 Refined
原版比例微调：外框圆角 7px，内核圆角 4px，间距 8px。更协调的视觉比例。
baseline
17-B
三层嵌套 Triple
三层结构：外框→中框→内核。精确对应 AST → 语义 → 渲染三层抽象。
三层抽象
17-C
同心圆 Circles
圆形版本。外环+内核。更柔和、更包容的嵌套表达。
ref: Target, Orbit
17-D
偏心嵌套 Offset
内核向右偏移 6px。打破对称，制造"流动/转化"的方向感。内核像在向外框右侧移动。
ref: Motion, Transformation
17-E
方圆嵌套 Hybrid
外方内圆。方形=代码结构，圆形=视觉图表。嵌套即转化——结构包裹着视觉。
ref: Morph + Nested
17-F
深度渐变 Depth
三层实心，透明度递增（20%→50%→100%）。像从远到近聚焦，内核最实。
ref: Z-depth, Focus
17-G
宽距嵌套 Wide Gap
更大的外框+更小的内核。留白更多，呼吸感更强。极简且精致。
ref: Airy, Minimal
17-H
双描边 Outline
两层都是描边，无填充。最通透、最克制的版本。轻盈如纸。
ref: Wireframe, Blueprint
17-I
框+点 Dot Core
外框+中心圆点。最极简的嵌套——框架包裹着一个语义核心。点即一切。
ref: Seed, Core
17-J
六边嵌套 Hexagon
六边形嵌套。外框是图表节点的经典形状，内核是语义核心。与产品领域强关联。
ref: Honeycomb, Node
